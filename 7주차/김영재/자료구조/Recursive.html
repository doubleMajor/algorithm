재귀 함수의 정의

  • 함수 안에 자신의 함수를 다시 호출하는 함수
  • 자신의 로직을 내부적으로 반복하다가, 일정한 조건이 만족되면 함수를 이탈하여 결과를 도출
  • 재귀 호출을 너무 많이 하게 되면 스택 메모리 영역에 너무 많은 공간을 할당하게 되어 스택 오버플로우가 발생할 수 있다.
  • 기본적으로, 함수가 호출 될 때 공간이 할당되고 해당 함수가 종료 될 때 해제 되는 메모리 영역을 Call Stack으로 나타낸다.
  • 함수가 호출 될 때마다, Call Stack에 메모리가 할당되며(스택 프레임이라고 한다.) 재귀 호출이 종료되는 시점에서 순서대로 함수를 실행한다. 
                    function recursive() {
                      call(0);
                    }
                    function call(index) {
                      if(index == 10) {
                          return;
                      }
                      call(++index);
                      console.log(index);
                    }

재귀 함수 연습 1 - 팩토리얼

  • 팩토리얼 계산을 재귀 함수로 구현해보자.
  • 팩토리얼을 수식으로 나타내면 
    N! = N*(N-1)*(N-2)*....*1
    이고, 0!, 1!의 값은 항상 1이 되기로 약속되어 있다.
  • 이러한 수식을 구현하기 위해 n이 1보다 클 경우 n * (n-1)...을 반복적으로 해주고 1보다 같거나 작을 경우 1을 반환하는 함수를 구현한다.
  • 아래 함수에서 factorial(3)을 구한다고 하면, 해당 함수는 
                                3 * factorial(3-1) =>
                            3 * 2 * factorail(2-1) =>
                            3 * 2 * 1 => 
                            결과 : 6
    이 된다. 
                    function recursive() {
                      factorial(number);
                    }
                    function factorial(n) {
                      if(n <= 1) {
                          return 1;    
                      } 
                      return n * factorial(n - 1);
                    }

재귀 함수 연습 2 - 피보나치 수열

  • N번째 피보나치 수열 재귀 함수를 구현해 구해보자.
  • 피보나치 수열의 수식은 
    f(n) = (fn-1) + f(n-2)
    이고, f(0), f(1)은 각각 0과 1이다.
  • 이러한 수식을 구현하기 위해, f(0)부터 f(n-1)과 f(0)부터 f(n-2)를 더하는 함수를 구현한다.
  • 아래 함수에서 fibonacci(3)을 구한다고 하면, 해당 함수는 
                                fibonacci(3-1) + fibonacci(3-2) =>
                            (fibonacci(2) => fibonacci(2-1) + fibonacci(2-2)) + (fibonacci(1)) =>
                            (fibonacci(1) + fibonacci(0)) + 1 => 
                            (1 + 0) + 1 => 
                            결과 : 2
    이 된다. 
                    function recursive() {
                      fibonacci(number);
                    }
                    function fibonacci(n) {

                      } 
                      return fibonacci(n - 1) + fibonacci(n - 2);
